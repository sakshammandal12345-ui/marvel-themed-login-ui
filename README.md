# marvel-themed-login-ui
A responsive, Marvel-themed login page featuring a modern design, social login options, and a square-centered layout.
## Features

-   **Marvel Theme**: Includes greeting and styling inspired by the Marvel universe.
-   **Responsive Design**: Works on various screen sizes.
-   **Square Layout**: Perfectly centered 1:1 login container.
-   **Social Login**: Buttons for Google and GitHub access (UI only).
-   **Animations**: Smooth fade-in entrance.

## Usage

1.  Clone the repository.
2.  Open `index.html` in your web browser.

## Technologies Used
-   HTML5
-   CSS3
-   JavaScript (Vanilla)
